# Smart Traffic Monitoring Dashboard 🚦

A desktop-based real-time traffic analysis system powered by **Deep Learning** and **Computer Vision**. This project automates vehicle classification and congestion detection to streamline urban traffic management.

## 🚀 Key Features
* **Real-Time Classification:** CNN model classifies vehicles with **92% accuracy**.
* **Automated Alerts:** Detection system reduces manual monitoring efforts by **40%**.
* **Responsive Dashboard:** Built with Tkinter for high-frequency data visualization.
* **Scalable Architecture:** Designed to handle continuous data input for real-world environments.

## 🛠️ Tech Stack
* **Language:** Python
* **GUI:** Tkinter
* **Deep Learning:** TensorFlow / Keras (CNN Architecture)
* **Computer Vision:** OpenCV
* **Data Processing:** NumPy, Pandas
